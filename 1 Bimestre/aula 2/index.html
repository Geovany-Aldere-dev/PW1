    Design Digital

    Nessa habilidade nós aprendemos sobre a parte gráfica do nosso site, como usar programas de edição de imagem e a criação de elementos visuais em que a gente possa interagir.

    Programação Web 1

    Nessa habilidade os alunos aprendem o desenvolvimento, nela aprendemos o HTML, CSS e Javascript que são utilizadas para a estrutura, interatividade e estilo.

    Técnicas de Programação

    Nela aprendemos os fundamentos da computação, algoritmos e a manipulação de dados e assim buscando a lógica e compreensão.

    Banco de Dados

    O Banco de Dados é o fundamento da informação, onde aprendemos a linguagem SQL, o gerenciamento e o armazenamento.

    Fundamentos da Informática

    Nela nós aprendemos o básico sobre os programas computacionais como; Software, hardware, sistemas operacionais e os conceitos da programação.

    Análise e Projetos de Sistemas

    Nela criamos projetos, escolhas de modelos para algum sistema, coleta de requisitos de usuários e sistemas, procedimentos de pesquisa e a coleta de dados.
